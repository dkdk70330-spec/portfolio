# AI 캐릭터 포트폴리오 1단계 프로토타입

백엔드 없이 실행되는 개인용 정적 포트폴리오입니다.

## 실행

가장 간단한 방법은 `index.html`을 더블클릭하는 것입니다.
브라우저 보안 설정에 따라 로컬 파일의 일부 동작이 제한되면 아래 명령으로 로컬 서버를 실행하세요.

```bash
python -m http.server 8000
```

그 후 브라우저에서 `http://localhost:8000`을 엽니다.

## 데이터 수정

`assets/data.js`의 `window.PORTFOLIO_DATA.characters` 배열을 수정합니다.

캐릭터마다 다음 항목을 가집니다.

- 이름과 한 줄 소개
- 상세 설명
- 장르와 태그
- 이미지 여러 장
- 플랫폼별 외부 링크
- 스포일러 경고와 후일담
- 대표 캐릭터 여부

## 이미지 교체

`images` 폴더에 이미지를 넣고 `assets/data.js`의 파일명을 변경합니다.
현재 이미지는 프로토타입용 SVG 플레이스홀더입니다.

## 포함 기능

- 제작자 프로필과 외부 링크
- 대표 캐릭터 영역
- 캐릭터 10명
- 이름·소개·태그 검색
- 장르 필터
- 플랫폼 필터
- 반응형 캐릭터 카드
- 상세 모달
- 이미지 갤러리
- 플랫폼 이동 링크
- 스포일러·후일담 접기/펼치기
- 밝은/어두운 테마
